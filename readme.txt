== Description ==

Converts the typed input into Indic Unicode.

Just activate the plugin, and you will see the effect in your comment box in front end

== Installation ==

You can either install it automatically from the wordpress admin panel or do it manually:

1. Unzip the archive and put the wp-unicode folder into your wordpress folder(/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.


== Usage ==

It do not have any admin end interface. You just need to go ahead and look over the comment box.
Start typing and see the changes.

Note that the changes can only be seen after you give a space and it requires internet connection.
